# YiffyAPI

<p>A hand picked image api, for furries.</p>

YiffyAPI consists of a handpicked [[Image API|YiffyAPI/Images]], a [[URL Shortener|YiffyAPI/Shortener]], an [[E621 Video Thumbnailer|YiffyAPI/Thumbnails]], and [[Meme Generation|YiffyAPI/ImageGen]].<br>
While many routes do not require an api key, using one is recommended. Getting one is free, and only requires signing in with Discord.<br>
You can manage your API keys here: <https://yiff.rest/apikeys>.
